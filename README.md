# CI/CD Pipeline using Jenkins

## 📌 Project Overview
This project demonstrates a CI/CD pipeline using Jenkins integrated with GitHub.

## 🚀 Tools Used
- Jenkins
- GitHub
- Docker

## ⚙️ Workflow
1. Code pushed to GitHub
2. Jenkins pulls code
3. Build triggered
4. Deployment done automatically

## 📂 Features
- Continuous Integration
- Continuous Deployment
- Automation using Jenkins
